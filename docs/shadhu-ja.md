# Shadhu Bhashaと現代ベンガル語：第1章の精読

本書は『বৌ-ঠাকুরাণীর হাট』(ボウ・タクラニル・ハート) 第1章の古典ベンガル語 (Shadhu Bhasha) を題材に、現代標準ベンガル語 (Cholito Bhasha) との違いを本文の実例で解説します。二つの文体の概説は [bhasha-ja.md](bhasha-ja.md) を参照してください。章全体を扱うのではなく、主要かつ体系的な違いを最もよく示す箇所を選んで取り上げます。

ローマ字表記はバッククォート内に、姉妹文書と同じ実用方式で示します。サンスクリットの語源は IAST (例: *rātri*)、ヒンディー語の対応語はデーヴァナーガリー文字と翻字で示します。

## 1. 動詞の活用 — 最大の違い

最も広範囲に及ぶ違いは動詞の語尾です。Shadhu Bhashaはほとんどの活用形で語中の母音 (-ই- `-i-` / -ইয়া `-iya`) を保持しますが、Cholito Bhashaはこれを脱落させ、その際に多くの場合、語幹の母音が変化します (例: করি- `kori-` → কর- `kor-`、যাই- `jai-` → যে- `je-`)。文法そのものは同一で、変わるのは語形だけです。

以下の例はすべて第1章に登場します。

| 種類 | Shadhu (本文) | Cholito | 意味 |
| :--- | :--- | :--- | :--- |
| 現在完了 | হইয়াছে (`hoiyachhe`) | হয়েছে (`hoyechhe`) | 〜になった |
| 現在完了 | শুনিয়াছি (`shuniyachhi`) | শুনেছি (`shunechhi`) | 私は聞いた |
| 現在進行 | নড়িতেছে (`noritechhe`) | নড়ছে (`norchhe`) | 動いている |
| 現在進行 | দেখিতেছেন (`dekhitechhen`) | দেখছেন (`dekhchhen`) | (彼が)見ている |
| 単純過去 | লাগিল (`lagilo`) | লাগল (`laglo`) | 〜し始めた |
| 単純過去 | চলিলাম (`cholilam`) | চললাম (`chollam`) | 私は出かけた |
| 未来 | আসিবে (`ashibe`) | আসবে (`ashbe`) | 来るだろう |
| 未来 | পারিব (`paribo`) | পারব (`parbo`) | 私はできるだろう |
| 過去習慣 | যাইতাম (`jaitam`) | যেতাম (`jetam`) | 私はよく行った |
| 不定詞 | বলিতে (`bolite`) | বলতে (`bolte`) | 言うこと |
| 接続分詞 | করিয়া (`koriya`) | করে (`kore`) | 〜して |
| 接続分詞 | বসিয়া আছেন (`boshiya achhen`) | বসে আছেন (`boshe achhen`) | 座っている |
| 完了否定 | হয় নাই (`hoy nai`) | হয়নি (`hoyni`) | 起こらなかった |
| 否定命令 | যাইও না (`jaio na`) | যেও না (`jeo na`) | 行かないで |

覚えておくべきパターン：

- **-িয়া `-iya` → -ে `-e`**: 接続分詞は短縮され、語幹母音が上がります (ধরিয়া `dhoriya` → ধরে `dhore`、বলিয়া `boliya` → বলে `bole`。不規則形として গিয়া `giya` → গিয়ে `giye`)。
- **-িতে- `-ite-` の -i- が脱落**: 進行形 করিতেছে `koritechhe` → করছে `korchhe`、不定詞 করিতে `korite` → করতে `korte`。母音語幹の動詞ではウムラウトが起きます: যাইতে `jaite` → যেতে `jete`。
- **-িব/-িল `-ibo/-ilo` の -i- が脱落**: 未来 করিব `koribo` → করব `korbo`、過去 করিল `korilo` → করল `korlo`。
- **完了否定**: Shadhuでは動詞の後に独立した語 নাই `nai` を置きますが、Cholitoでは接尾辞 -নি `-ni` を付けます。

## 2. 代名詞 — 母音間の -হ- (-h-) の脱落

Shadhuの三人称代名詞・関係代名詞には -হ- `-h-` が含まれますが、Cholitoではこれが脱落して語が縮まります。この変化は完全に規則的です。

| Shadhu (本文) | Cholito | 意味 |
| :--- | :--- | :--- |
| তাঁহার (`tãhar`) | তাঁর (`tãr`) | 彼/彼女の (敬称) |
| তাহার (`tahar`) | তার (`tar`) | 彼/彼女の |
| তাহারা (`tahara`) | তারা (`tara`) | 彼ら |
| যাহারা (`jahara`) | যারা (`jara`) | 〜する者たち |
| ইহার (`ihar`) | এর (`er`) | これの |
| উহার (`uhar`) | ওর (`or`) | それの/あの人の |
| তাহা (`taha`) | তা (`ta`) | それ |
| কেহ (`keho`) | কেউ (`keu`) | 誰か |
| কাহাকেও (`kahakeo`) | কাউকে (`kauke`) | 誰に(で)も |

本文からの例 (夫を擁護するスルマの台詞)：

> তােমাকে **যাহারা** নির্ব্বোধ মনে করিত **তাহারাই** নির্ব্বোধ!
> `tomake jahara nirbodh mone korito taharai nirbodh!`
> Cholito: তোমাকে **যারা** নির্বোধ মনে করত **তারাই** নির্বোধ!
> 「あなたを愚か者だと思った者たちこそが愚か者です！」

## 3. 後置詞と機能語

| Shadhu (本文) | Cholito | 意味 | 備考 |
| :--- | :--- | :--- | :--- |
| হইতে (`hoite`) | থেকে (`theke`) | 〜から | হইতে は動詞 হ-「〜になる」の文法化、থেকে は থাকা「留まる」由来。ヒンディー語は無関係の से (`se`) を使う |
| অপেক্ষা (`opekkha`) | চেয়ে (`cheye`) | 〜より | Tatsama語。Skt. *apekṣā*「期待、考慮」。フォーマルなヒンディー語の की अपेक्षा (`kī apekṣā`)「〜と比べて」と全く同じ構文 |
| নিকটে (`nikote`) | কাছে (`kachhe`) | 〜の近くに、〜のもとへ | Tatsama語。Skt. *nikaṭa*。ヒンディー語 निकट (`nikaṭ`) はフォーマルな「近く」として現存 |
| দ্বারা (`dbara`) | দিয়ে (`diye`) | 〜によって | Tatsama語。Skt. *dvārā* (「扉」の具格、原義「扉を通って」)。ヒンディー語 द्वारा (`dvārā`) はフォーマルな文章で今も標準的。দ্বারা はフォーマルなCholitoにも残る |
| ত (`to`) | তো (`to`) | 強調の小詞 | ヒンディー語の तो (`to`) と同じ小詞。綴りが変わっただけ |

本文からの例 (父の監視について語る王子の台詞)：

> পিতা ছেলেবেলা **হইতেই** আমাকে প্রতি মুহূর্ত্তে পরখ করিয়া দেখিতেছেন
> `pita chhelebela hoitei amake proti muhurte porokh koriya dekhitechhen`
> Cholito: বাবা ছেলেবেলা **থেকেই** আমাকে প্রতি মুহূর্তে পরখ করে দেখছেন
> 「父は私が幼い頃からずっと、一瞬一瞬私を試すように見てきた」

## 4. 語彙 — Tatsama対Tadbhava、ヒンディー語の対応語とともに

Shadhu Bhashaはサンスクリットからの直接借用語 (**Tatsama語**) を好み、Cholitoは自然に変化した形 (**Tadbhava語**) や借用語を好みます。注目すべきパターンとして、Shadhuベンガル語がTatsama語を使うところで、日常ヒンディー語は同じサンスクリット語の*Tadbhava形*を使っていることが多く、その結果、現代口語ベンガル語と日常ヒンディー語はしばしば同源の形に行き着きます (হাত/हाथ、রাত/रात、ঘর/घर)。

以下のShadhu語はすべて第1章に登場します。

| Shadhu (本文) | Cholito | 意味 | 語源 | ヒンディー語 |
| :--- | :--- | :--- | :--- | :--- |
| রাত্রি (`ratri`) | রাত (`rat`) | 夜 | Tatsama語、Skt. *rātri*。রাত はそのTadbhava形 | रात्रि (`rātri`、フォーマル) / রাত = रात (`rāt`、日常語) |
| হস্ত (`hosto`) | হাত (`hat`) | 手 | Tatsama語、Skt. *hasta*。হাত はそのTadbhava形 | हस्त (`hast`、フォーマル) / हाथ (`hāth`、日常語) |
| দক্ষিণ (`dokkhin`) | ডান (`ḍan`) | 右(の手) | Tatsama語、Skt. *dakṣiṇa*。Cholitoの দক্ষিণ は「南」の意味でのみ現存 | दाहिना (`dāhinā`) は同じ語のTadbhava形。दक्षिण は「南」 |
| চক্ষু (`chokkhu`) | চোখ (`chokh`) | 目 | Tatsama語、Skt. *cakṣus*。চোখ はそのTadbhava形 | 日常語 आँख (`ā̃kh`) は別のSkt.語 *akṣi* に由来。चक्षु は詩語 |
| বাতায়ন (`batayon`) | জানালা (`janala`) | 窓 | Tatsama語、Skt. *vātāyana*「風の通り道」。জানালা はポルトガル語 *janela* 由来 | खिड़की (`khiṛkī`)。詩語として वातायन (`vātāyan`) |
| শয়ন-গৃহ (`shoyon-griho`) | শোবার ঘর (`shobar ghor`) | 寝室 | 両要素ともTatsama語: Skt. *śayana*「横たわること」+ *gṛha*「家」 | शयनकक्ष (`śayan-kakṣ`、フォーマル)。ঘর `ghor` と घर (`ghar`) はともに *gṛha* のTadbhava形 |
| পার্শ্বে (`parshe`) | পাশে (`pashe`) | 傍らに | Tatsama語、Skt. *pārśva*。পাশ はそのTadbhava形 | पार्श्व (`pārśva`、フォーマル)。日常語 पास (`pās`)「近く」は同じTadbhava形 |
| কহিলেন (`kohilen`) | বললেন (`bollen`) | 言った | কহ্- はSkt. √*kath* (*kathayati*「語る」) 由来 | कहना (`kahnā`) はヒンディー語の通常の動詞 — ここではShadhuベンガル語がヒンディー語と一致し、Cholitoは বলা `bola` に切り替わった |
| বৎসর (`botsor`) | বছর (`bochhor`) | 年 | Tatsama語、Skt. *vatsara*。বছর はそのTadbhava形 | वर्ष (`varṣ`、フォーマル) / साल (`sāl`、ペルシャ語) / बरस (`baras`、*varṣa* のTadbhava形) |
| জ্যেষ্ঠ পুত্র (`jyeshtho putro`) | বড় ছেলে (`boro chhele`) | 長男 | Tatsama語、Skt. *jyeṣṭha putra* | ज्येष्ठ पुत्र (`jyeṣṭh putra`、フォーマル) / बड़ा बेटा (`baṛā beṭā`)。বড় と बड़ा は同源 |
| ভগিনী (`bhogini`) | বোন (`bon`) | 姉妹 | Tatsama語、Skt. *bhaginī*。বোন はそのTadbhava形 | बहन (`bahan`) — これも *bhaginī* のTadbhava形で、বোন と बहन は同源 |
| দুয়ার (`duyar`) | দরজা (`doroja`) | 扉 | দুয়ার 自体がSkt. *dvāra* のTadbhava形 — Shadhuの散文にもTadbhava語は混じる。দরজা はペルシャ語 *darwāza* 由来 | द्वार (`dvār`、フォーマル) / दरवाज़ा (`darvāzā`) — দরজা と同じペルシャ語借用 |
| পিতা (`pita`) | বাবা (`baba`) | 父 | Tatsama語、Skt. *pitṛ* | पिता (`pitā`) はヒンディー語では今も普通の丁寧な語。ベンガル語は話し言葉で বাবা に移行した |

なお জল `jol`「水」(সুরমার চখে জল আসিল「スルマの目に涙が浮かんだ」) は西ベンガルでは今も標準語ですが、バングラデシュでは পানি `pani` が好まれます — これはヒンディー語の पानी (`pānī`) と同源です。

## 5. 正書法

純粋に綴り字改革による違いもあります — 発音はもともと同じでした。

- **র্ (レフ) の後の子音重複**: 19世紀の印刷では、サンスクリット文法が任意で認めていたとおり、র্ に続く子音を重ねて綴りました。カルカッタ大学の綴り字改革 (1936年) でこれは廃止されました。本章から: ধৈর্য্য → ধৈর্য (`dhoirjo`)、কার্য্য → কার্য (`karjo`)、মুহূর্ত্ত → মুহূর্ত (`muhurto`)、পূর্ব্ব → পূর্ব (`purbo`)、নির্ব্বোধ → নির্বোধ (`nirbodh`)、সর্ব্বনাশ → সর্বনাশ (`shorbonash`)、আশীর্ব্বাদ → আশীর্বাদ (`ashirbad`)、কর্ম্মচারী → কর্মচারী (`kormochari`)。
- **কি と কী**: 古いテキストは諾否疑問の小詞と疑問詞「何」の両方を কি と綴りますが、現代の正書法はこれを区別します。কি হইয়াছে? → কী হয়েছে? (`ki hoiyachhe → ki hoyechhe`)「何があったの？」
- **জিনিষ → জিনিস** (`jinish`)「物」: かつて ষ で綴られた借用語は現在 স を使います。
- 原文ファイルについての注意: `wikisource/chapters/01.txt` の যশােহর のような綴りは、母音記号 ো が া + ে の2文字でエンコードされたもので、デジタル化の際のアーティファクトであり、歴史的な綴りの違いではありません。

## 6. 実例の精読

**A. スルマの慰め (この章のリフレイン)**

> Shadhu: সুরমা কহিলেন, "প্রিয়তম, সহ্য করিয়া থাক, ধৈর্য্য ধরিয়া থাক। এক দিন সুখের দিন আসিবে।"
> `Surama kohilen, "priyotomo, shojjho koriya thako, dhoirjo dhoriya thako. ek din shukher din ashibe."`
> Cholito: সুরমা বললেন, "প্রিয়তম, সহ্য করে থাকো, ধৈর্য ধরে থাকো। একদিন সুখের দিন আসবে।"
> 「スルマは言った。『愛しい人、耐えてください、辛抱してください。いつか幸せな日が来ます』」

この一文だけで4種類の変化が見られます: 発話動詞 (কহিলেন → বললেন)、2つの接続分詞 (করিয়া → করে、ধরিয়া → ধরে)、未来形 (আসিবে → আসবে)、綴り字改革 (ধৈর্য্য → ধৈর্য)。

**B. ルクミニについて語る王子**

> Shadhu: রুক্মিণীর বয়স আমার অপেক্ষা তিন বৎসরের বড়। ... ইহার পূর্ব্বেও আমার এমন কখন হয় নাই।
> `Rukminir boyosh amar opekkha tin botsorer boro. ... ihar purbeo amar emon kokhon hoy nai.`
> Cholito: রুক্মিণীর বয়স আমার চেয়ে তিন বছরের বড়। ... এর আগেও আমার এমন কখনো হয়নি।
> 「ルクミニは私より三つ年上だった。……あんなことはそれまで一度も起きたことがなかった」

ここでは比較の後置詞 (অপেক্ষা → চেয়ে)、Tatsama名詞 (বৎসর → বছর)、代名詞 (ইহার → এর)、Tatsamaの副詞表現 (পূর্বে → আগে「以前に」)、完了否定 (হয় নাই → হয়নি) がすべて変化します。

**C. 扉口のビバ**

> Shadhu: বিভা বলিয়া উঠিল "না না তুমি যাইও না।" ... "পিতা যদি জানিতে পারেন? তােমার উপরে যদি রাগ করেন?"
> `Bibha boliya uthilo "na na tumi jaio na." ... "pita jodi janite paren? tomar upore jodi rag koren?"`
> Cholito: বিভা বলে উঠল "না না তুমি যেও না।" ... "বাবা যদি জানতে পারেন? তোমার ওপরে যদি রাগ করেন?"
> 「ビバは叫んだ。『だめ、だめ、行かないで！』……『お父様が知ったら？　あなたに怒ったら？』」

## 7. 最後の観察: 台詞に漏れ出すCholito

この小説 (1883年) は全編Shadhu Bhashaで書かれていますが、タゴールは感情が高ぶる台詞に口語形を滑り込ませています。怯えたビバが দাদা কি হবে? (`dada ki hobe?`「兄さん、どうなるの？」)、কি হবে ভাই? বাবা যদি টের পান? (`ki hobe bhai? baba jodi ter pan?`) と叫ぶとき、彼女はCholitoの未来形 হবে (Shadhuの হইবে ではなく) と口語の বাবা (地の文のTatsama語 পিতা ではなく) を使っています。登場人物はあらたまった述懐ではShadhuで語りますが、切迫した場面では現実の話し言葉に近づくのです — これは1910年代以降にタゴール自身が文学語としてのCholito Bhashaを推進することの予兆とも言えます。
